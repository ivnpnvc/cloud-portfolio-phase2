Cloud Architecture Demo Page

Diese statische Website dient als technischer Demonstrator für eine einfache, reproduzierbare und weltweit erreichbare Bereitstellung in Microsoft Azure.

Projektziel

Im Mittelpunkt steht nicht die Webentwicklung, sondern die Cloud-Architektur. Die Seite zeigt, wie eine statische Anwendung mit Azure Storage, Azure Front Door und Terraform bereitgestellt werden kann.

Technische Informationen

  • Cloud-Anbieter: Microsoft Azure
  • Bereitstellung: Infrastructure as Code mit Terraform
  • Anwendungstyp: Statische Website
  • Sicherheitsansatz: HTTPS, keine Nutzerdaten, keine Formulare

Hinweis zum Demonstrator

Es werden keine Benutzereingaben verarbeitet, keine Daten gespeichert und keine serverseitigen Geschäftslogiken eingesetzt. Dadurch bleibt die Lösung bewusst einfach und konzentriert sich auf die Anforderungen des Portfolios.
