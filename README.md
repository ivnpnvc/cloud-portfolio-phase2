# cloud-portfolio-phase2


Dieses Projekt enthält die Terraform-Konfiguration zur Bereitstellung einer statischen Website in Microsoft Azure.

Inhalt:
- Infrastructure as Code mit Terraform
- Azure Storage Static Website
- einfache HTML/CSS/JS Anwendung

Deployment:
terraform init
terraform apply

Hinweis:
Azure Front Door war ursprünglich geplant, konnte aber aufgrund der verwendeten Subscription nicht eingesetzt werden.
